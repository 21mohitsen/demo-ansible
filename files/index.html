Tips

  • Start with compute & storage for web apps: EC2 + S3.
  • Use serverless (Lambda) to avoid managing servers for small workloads.
  • Use IAM roles and KMS for secure access and key management.
  • For analytics use Athena, Redshift or EMR depending on scale.

FAQ

What is the difference between EC2 and Lambda?

EC2 provides virtual servers with full control; Lambda is serverless and charges per execution with automatic scaling.

When to choose RDS vs DynamoDB?

RDS suits relational workloads with complex queries; DynamoDB is a fully managed NoSQL database for single-digit millisecond performance at scale.
